Golang中国社区成长时间线

  • 2015

  • 实现 Markdown 编辑器的图片上传功能

    2015-05-09

    通过 editor.md 的上传插件，上传图片到服务器后再传到七牛。

  • Markdown 编辑器换成 editor.md

    2015-05-04

    editor.md 是一款国人开发的 Markdown 编辑器，默认支持 GFM 。

  • 2014

  • 增加第三方包下载功能

    2014-07-23

    可以下载 Google Code 等不能访问的代码托管网站上的 package 。

  • 2013

  • 增加图书板块

    2013-11-06

    列出 Go 语言相关的中英文图书。

  • 网站域名由 golang.tc 换为 golangtc.com

    2013-09-03

    golang.tc 域名悲催地被 Godaddy 没收，启动新域名，从头再来。

  • 增加下载功能

    2013-08-23

    由于 Go 官方下载不稳定，提供 Go 安装包和源码的本地下载。

  • 增加友情链接功能

    2013-07-23

  • 增加查看同城会员功能

    2013-04-09

  • 用户头像使用七牛云存储加速

    2013-04-05

    一开始用户头像适用的 Gravatar 服务，但是国内访问非常慢，改用七牛存储用户头像。

  • 增加浏览所有会员功能

    2013-03-27

  • 增加搜索主题功能

    2013-03-21

    随着主题数量也来越多，想要一页页翻找感兴趣的主题已经变得困难，因此加上了搜索功能，可以根据标题和内容进行搜索。

  • 增加查看新加入会员功能

    2013-03-19

  • 第三方包增加评论功能

    2013-03-13

  • 静态文件放到七牛云存储，提高访问速度

    2013-02-20

  • 增加第三方包板块

    2013-01-22

    用于提交第三方包的介绍，网址，方便用户查找第三方包。

  • 2012

  • 增加文章板块

    2012-12-12

    用于转载文章。

  • 增加 Go 指南功能

    2012-09-28

    对https://go-tour-zh.appspot.com/做反向代理，免去翻墙。

  • 代码移至Github开源

    2012-09-01

    代码地址https://github.com/jimmykuu/gopher 。

  • Golang 中国社区启动，社区和酷站功能

    网址 golang.tc 。
